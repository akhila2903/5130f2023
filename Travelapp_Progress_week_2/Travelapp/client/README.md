# TravelApp - Frontend

## Description

React Frontend for TravelApp application, where user can view all available places, register, login and book.

## Usage

1. Clone the repo

```bash
    git clone 
    cd TravelApp/client
```

2. Install dependencies

```bash
    npm install
```

3. Run

```bash
    npm run dev
```

Open `http://127.0.0.1:5173` to view the website
